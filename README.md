# 🚀 Elaboración de un Sistema de Biblioteca en Java
Uno de mis mayores retos fue desarrollar un sistema de forma independiente, ya que no había cursado una materia que debía haber llevado en un ciclo normal. Para ello, tuve que inscribirme en un interciclo y encontré a compañeros que nunca habían trabajado con Java ni base de datos, solo tenían conocimientos en C++. Dado que esta materia se impartía casi en la mitad de la carrera, me propuse hacer el sistema de forma independiente, para pasar la materia y aprender nuevas cosas por mi cuenta.
<br><br>
Tuve que explicar a mi grupo cómo funcionaban los diagramas de flujo, el modelo de la base de datos, los diagramas de casos de uso y los diferentes escenarios que podrían preguntarnos durante la defensa grupal e individual. Durante el transcurso del proyecto, ayudé a mis compañeros a comprender estos conceptos y a guiarlos en su implementación. Algunos captaron rápidamente y me ayudaron, mientras que otros necesitaron más apoyo.
